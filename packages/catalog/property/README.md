## Configuration Options

| Option | Type | Default | Description |
| ------ | ---- | ------  | ----------- |
|`schema` | [JSON schema object](http://json-schema.org/) | `{}` | Describes the object and its properties |
| `initialObjects` | `Object[]` | `[]` | Seeds the property database |
