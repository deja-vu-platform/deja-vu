# Ranking

Rank items

## Components

- create-ranking
- show-fractional-ranking
- show-ranking
- show-rankings
- show-target

## Configuration Options

| Option | Type | Default | Description |
| ------ | ---- | ------  | ----------- |
| `oneToOneRanking` | `boolean` | `false` | Determines whether every `sourceId` can only rank a `targetId` once in the global set of rankings. |
